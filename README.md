## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

1. Feat - 새로운 기능 추가
2. Fix - 버그 수정
3. Docs - 문서 추가 및 수정
4. Chore - 빌드 업무 수정, 패키지 매니저 수정 (소스 코드 변경 X)
5. Design - CSS 등 사용자 UI 디자인 변경
6. Remove - 파일을 삭제하는 작업만 수행한 경우
7. Rename - 파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우
8. Refactor - 소스 코드 리팩토링
